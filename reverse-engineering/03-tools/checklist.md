# Checklist: Tools Mastery

## Disassemblers
- [ ] **IDA Pro:** Navigation, Cross-references (Xrefs), Renaming variables (N), Comments (;).
- [ ] **Ghidra:** Decompiler view, Project management.

## Debuggers
- [ ] **x64dbg:** Attach to process, Software Breakpoints (F2), Hardware Breakpoints.
- [ ] **GDB (Linux):** break, step i, continue, examine memory (x/...).

## Utilities
- [ ] **Process Hacker/Explorer:** Inspect strings in memory, check loaded modules.
- [ ] **Wireshark:** Follow TCP Stream, Capture filters.
